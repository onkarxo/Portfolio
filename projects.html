Interactive Spell Checker

Language Used: Python
The Interactive Spell Checker is a Python-based application utilizing the Tkinter library to create a user-friendly interface for spell checking and also represents an application of Natural Language Processing (NLP) principles. It leverages the Enchant library to identify and suggest corrections for misspelled words in real-time. Users can input text into the interface, and the application automatically detects and highlights misspelled words. It provides suggestions for corrections upon hovering over highlighted words, allowing users to replace them with the suggested alternatives. Additionally, it includes a 'Correct Spelling' button to instantly correct all identified misspellings within the text.
